Materials and Constructions — Curated Overview

This local page is a curated helper for Ray-Modeler. It focuses on the EnergyPlus objects used by the Materials and Constructions configuration panels. For full details, consult the official EnergyPlus Input Output Reference.

Material

Material defines an opaque layer with thermal mass.

  • Thickness — Layer thickness [m].
  • Conductivity — Thermal conductivity [W/m·K].
  • Density — Density [kg/m³].
  • Specific Heat — Heat capacity [J/kg·K].
  • Solar/Visible Absorptance — Optional surface properties.

Ray-Modeler "opaque material" entries map directly to Material. Values you set here are used when assembling wall/roof/floor constructions.

Material:NoMass

Material:NoMass represents a purely resistive layer with no heat capacity.

  • Thermal Resistance — R-value [m²·K/W].
  • Solar/Visible Absorptance — Optional.

In Ray-Modeler, "no-mass" materials map to Material:NoMass and are typically used for insulation or air films described only by R-value.

Material:AirGap

Material:AirGap represents a fixed thermal resistance for an air gap.

Ray-Modeler may represent air gaps using either Material:NoMass with an R-value or Material:AirGap. Consult the builder’s mapping if applicable.

WindowMaterial:* (Glazing and Layers)

Window layers use the WindowMaterial:* family. A common simplified form:

WindowMaterial:SimpleGlazingSystem

  • U-Factor — Overall U-value [W/m²·K].
  • SHGC — Solar Heat Gain Coefficient [-].
  • Visible Transmittance — Fraction of visible light transmitted.

Ray-Modeler "simple glazing" entries map to WindowMaterial:SimpleGlazingSystem. Use these fields when defining custom window constructions.

Construction

Construction defines the ordered stack of material layers for an opaque or window assembly.

  • Layers are listed from outside to inside.
  • Each layer references a Material or WindowMaterial:* by name.

In Ray-Modeler, a "Construction" you define is written as an EnergyPlus Construction object. The Materials Manager exposes the materials that can be used in these stacks.

Construction:InternalSource

Construction:InternalSource extends Construction to model internal heat sources (e.g., embedded radiant systems) within a layer stack.

If Ray-Modeler adds support for embedded sources, it will map specialized constructions to Construction:InternalSource. For now, standard user constructions generally use Construction.

Mapping summary: - Ray-Modeler "Materials" panel → EnergyPlus Material / Material:NoMass / WindowMaterial:*. - Ray-Modeler "Constructions" panel → EnergyPlus Construction (layers refer to those materials). Ensure names are unique and consistent; constructions reference materials by name.
